# Tone presets

Pick one tone and keep it consistent across every piece of server text (name, MOTD, ranks, store,
Discord, in-game). Each preset fixes word choice, punctuation/emoji use, rank-name style, and a
palette leaning. Examples are illustrative — adapt to the actual server name.

## cozy-smp
- **Voice:** warm, welcoming, lowercase-friendly, first-person plural ("we", "our little world").
- **Punctuation/emoji:** soft, sparse emoji (🌿🏡✨); no ALL CAPS.
- **Rank names:** Guest → Member → Friend → Elder → Patron (gentle, community words).
- **Palette:** greens, warm browns, soft gold (`#7FB069 #A3C586 #E9C46A #B08968`).
- **Example MOTD line:** `<gradient:#A3C586:#E9C46A>Willow Hollow SMP</gradient> — pull up a chair`.

## hardcore-anarchy
- **Voice:** blunt, terse, no hand-holding; "no rules, no resets, no mercy".
- **Punctuation/emoji:** minimal, gritty; emoji rare (☠️⛓️); occasional ALL CAPS for impact.
- **Rank names:** avoid pay ranks framing; use Survivor / Veteran / Warlord (earned, not bought).
- **Palette:** blood red, ash, black (`#8B0000 #C1121F #2B2B2B #6E6E6E`).
- **Example MOTD line:** `<gradient:#8B0000:#C1121F>VOIDFALL</gradient> <dark_gray>// no resets. ever.`

## competitive-pvp
- **Voice:** sharp, energetic, stat- and rank-driven; verbs like "climb", "clutch", "dominate".
- **Punctuation/emoji:** crisp; ⚔️🏆🔥 sparingly; tight slogans.
- **Rank names:** Bronze → Silver → Gold → Diamond → Champion, or Recruit → Elite → Pro → Legend.
- **Palette:** electric blue, cyan, sharp white (`#1B6CA8 #2EC4F1 #5EE7DF #F1F5F9`).
- **Example MOTD line:** `<gradient:#2EC4F1:#5EE7DF>APEX Practice</gradient> <gray>| ranked + unranked`.

## kid-friendly
- **Voice:** bright, simple, encouraging; short words; positive framing of rules.
- **Punctuation/emoji:** plenty of friendly emoji (🎈🌈🐾); never edgy or sarcastic.
- **Rank names:** Explorer → Adventurer → Hero → Champion → Star (no skulls/weapons).
- **Palette:** rainbow-bright primaries (`#FF6B6B #FFD93D #6BCB77 #4D96FF`).
- **Example MOTD line:** `<rainbow>Bloomtopia</rainbow> <gray>a friendly place to build!`.

## premium-clean
- **Voice:** minimal, confident, understated; few words; no hype, no exclamation spam.
- **Punctuation/emoji:** almost no emoji; one accent glyph max (✦ •).
- **Rank names:** Member → Plus → Premium → Elite → Founder (clean tiers).
- **Palette:** charcoal, off-white, single accent gold/teal (`#14161B #E5E7EB #E0992E` or `#14B8A6`).
- **Example MOTD line:** `<#E0992E>AETHER</#E0992E> <dark_gray>•</dark_gray> <gray>refined survival`.

## meme-edgy
- **Voice:** irreverent, internet-native, self-aware; jokes welcome; don't take itself seriously.
- **Punctuation/emoji:** chaotic but readable; 💀🗿🔥; lowercase or rAnDoM as a bit (sparingly).
- **Rank names:** Peasant → Sweat → Tryhard → Goblin King → Certified Menace.
- **Palette:** lime, magenta, black (`#A3E635 #EC4899 #0F0F0F`).
- **Example MOTD line:** `<gradient:#A3E635:#EC4899>skibidi smp</gradient> <gray>0 lag (real)`.

## Cross-tone rules
- Keep the **name's** capitalization consistent everywhere (MOTD, ranks, Discord, store).
- Don't promise gameplay-affecting/pay-to-win store perks unless the user explicitly asks (EULA).
- Re-use the chosen palette for the MOTD gradient, rank prefixes, and scoreboard accents so the
  whole brand reads as one piece.
