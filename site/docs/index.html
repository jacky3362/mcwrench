Documentation

mcwrench is a Minecraft server config and operations assistant. It reads and advises, quotes real config keys, prioritises findings, and never invents keys. Start below, or jump to a section.

// principles

How it works

  • ▣A router plus specialists. The always-on minecraft-server-router engages on any server-admin topic and hands off to the right specialist.
  • ▣It never invents config keys or versions. When unsure about a plugin, it runs learn-plugin-docs to fetch and condense that plugin's real docs, then answers from them.
  • ▣Findings are prioritised: critical (crash or data loss), then high (lag or security), then medium, then low.
  • ▣Destructive actions are confirmed first (deleting worlds, exposing RCON, flipping online-mode behind a proxy).
Current facts (2026). Minecraft 26.1.x, Paper 26.1+ needs Java 25, Velocity 3.5.0-SNAPSHOT, Paper downloads via Fill v3. mcwrench confirms your actual version before version-specific advice.
