Changelog

Versioned with SemVer. The canonical log lives in CHANGELOG.md.

v1.0.0

latest 2026-06-07

First public release. A gamemode-agnostic Minecraft server config and operations assistant, packaged for Claude Code, Claude.ai, OpenAI Codex, Google Antigravity, and Gemini CLI from one source of truth.

Skills (10)

  • + minecraft-server-router, the always-on hub
  • + audit-config, with scan-server-tree and diff-against-defaults helpers
  • + performance-tuning (Spark, Aikar vs Java-25 ZGC, view and sim distance)
  • + permissions-helper, proxy-network, skript-author
  • + gamemode-stacks, 11 archetypes
  • + new-server-bootstrap, with a Fill v3 Paper downloader
  • + pterodactyl-ops (Pterodactyl and Pelican)
  • + learn-plugin-docs, the on-demand docs engine

Docs learner

Every adapter wired and live-tested: Modrinth, Hangar (slug-only API), GitHub README, Oraxen, SpigotMC (BBCode to markdown), Skript Hub, PaperMC (tree-searches the source repo). GitBook and a generic Readability fallback are best-effort.

Cross-tool packaging

One source of truth in skills/, with no duplicated SKILL.md. Claude Code plugin and marketplace; .agents/skills symlink, AGENTS.md, and workflows for Codex and Antigravity; GEMINI.md and TOML commands for Gemini CLI; a zip packer for Claude.ai.

Notes

Knowledge dated 2026-06-07: Minecraft 26.1.x, Paper requires Java 25, Velocity 3.5.0-SNAPSHOT, Paper downloads via Fill v3, Pelican is Pterodactyl-compatible (beta). The Aikar's-flags Java-25 caveat is attributed to community and hosting sources, not PaperMC.
