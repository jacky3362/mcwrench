Optional: to use the exact dafont "Minecraft" font by Craftron Gaming instead of Monocraft,
download it from https://www.dafont.com/minecraft.font and save the file here as:

    site/fonts/Minecraft.ttf

The page auto-detects it (CSS @font-face family "MinecraftCraftron") and uses it for the
wordmark/headings. If this file is absent, the page falls back to Monocraft (OFL 1.1) from a CDN.
