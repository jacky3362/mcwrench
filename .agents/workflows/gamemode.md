# Gamemode stack

1. Engage the **gamemode-stacks** skill (`.agents/skills/gamemode-stacks/SKILL.md`).
2. Identify the archetype (SMP, skyblock, prison, factions, towny, minigames/network, RPG/MMO,
   anarchy, creative, KitPvP, lifesteal) and read its reference.
3. Confirm server software/version and whether it's behind a proxy (networks → **proxy-network**).
4. For exact config keys/permissions of any plugin, run **learn-plugin-docs** first.
5. Greenfield? start with **new-server-bootstrap**. Review the result with **audit-config** and
   tune with **performance-tuning**.
