# Permissions (LuckPerms)

1. Engage the **permissions-helper** skill (`.agents/skills/permissions-helper/SKILL.md`).
2. Establish the rank ladder (groups + a track) before assigning individual nodes.
3. Scope per-world / per-server / per-gamemode perks with contexts (see `luckperms-contexts.md`).
4. Set group weights for clean prefix/suffix resolution.
5. Give the exact `/lp …` commands to run, in order. If a plugin's node names are uncertain, run
   **learn-plugin-docs** first — do not invent nodes.
